# SmartHome

## Описание
SmartHome — это проект умного дома, который позволяет управлять устройствами, такими как свет, термостаты и обогреватели, через консольное приложение.

## Требования
- .NET 7.0 или выше

## Установка
1. Убедитесь, что у вас установлен .NET 7.0 или выше. Вы можете проверить это, выполнив команду:
   ```sh
   dotnet --version
   ```
2. Склонируйте репозиторий или скопируйте файлы проекта в нужное место.

## Запуск
1. Перейдите в директорию проекта:
   ```sh
   cd SmartHome
   ```
2. Запустите приложение:
   ```sh
   dotnet run
   ```

## Использование
После запуска приложения, вы увидите главное меню с доступными опциями:
1. Показать все состояния
2. Управление комнатой
3. Управление всеми устройствами
4. Задать желаемые параметры дома
5. Задать текущие параметры
6. Сцены
7. Включить автоматизацию
8. Имитировать движение
0. Выход

Выберите нужную опцию, введя соответствующий номер и нажав Enter.
